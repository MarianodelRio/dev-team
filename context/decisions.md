# Decision Log

Technical decisions made during development. Every agent that makes a non-obvious choice writes here.

**Format:**
```
## YYYY-MM-DD — T-XXX [Agent name]
Decided: [what was decided]
Why: [the reason — constraint, trade-off, research finding]
Affects: [files or modules impacted]
Discarded: [alternative considered and why it was rejected]
```

The Orchestrator reads this file before starting each task and surfaces relevant entries in the human checkpoint.
